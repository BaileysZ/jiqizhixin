# 减肥辅助bot

- 作品介绍：应用场景
	-本产品的主要功能有两个：
		- 信息问询
			- 输入食物或者运动，输出食物对应的热量，以及运动对应会消耗的热量
		- 减肥咨询：
			- 通过用户输入身高体重，计算出静息代谢
			- 通过用户输入目标体重和现在的体重，计算出每天应该消耗多少热量，摄入多少热量
