1、应用场景：

在过去的几十年里，环境污染已经成为影响人们健康的一个主要问题，特别是那些发展中国家。与城市污染作斗争将需要进行全面的长期环境数据收集工作，这些工作主要依靠监测站。但监测点往往不足，难以获得全市细粒度的环境状态。

本作品采用了群智感知的思想，提出了一个空气质量监测系统的设计。该系统可以将智能手机变成高质量的PM2.5监测器，使用户可以随时随地利用收集摄像头进行空气质量的测量。
